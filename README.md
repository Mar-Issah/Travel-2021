# Travel 2021
This is a simple web application web application project mainly to practice my HTML5, CSS3 and JavaScript skills. 

## About The Project

![Travel 2021](https://res.cloudinary.com/dytnpjxrd/image/upload/v1617292175/My%20Website%20Projects/travel_2021_meiwch.png)

This app gives you a tip of which country to travel to in 2021. (Travel after Covid...lol). You can click on the previous or next button to view countries or you can click on the surprise me button which displays a random country.

## Getting Started
This is JavaScript Web application which runs in the browser. You need a V8 engive to run JavaScript program. For the purpose of this project I used Google Chrome browser.

### Installation

1. Clone the repo on your local machine
   ```sh
   git clone https://github.com/Mar-Issah/Travel-2021.git
   ```
2. Download and install [Visual Studio Code](https://code.visualstudio.com/)
3. Install live server extension
4. Open index.html file with live server to run the app in browser

# OR
5. Search for the index.html file in the folder and open/run with your browser
